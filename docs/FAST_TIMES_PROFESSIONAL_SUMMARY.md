## 📱 INSTAGRAM TEMPLATE SPECIFICATIONS

### Template A: EDUCATIVO
- **Purpose:** Teach, educate, build authority
- **Ratio:** 1080 x 1350px (vertical)
- **Layout:** Eyebrow + Gradient Headline + Body + Metric Grid
- **CTA:** "Guarda | Comenta tu zona"
- **Best for:** Mondays
- **Expected Engagement:** 150-300 likes, 50-100 comments

### Template B: OFERTA
- **Purpose:** Convert, sell, drive action
- **Ratio:** 1080 x 1350px (vertical)
- **Layout:** Red Header + 3 Plan Cards + Primary CTA Button
- **Featured Card:** Competitivo (most popular)
- **CTA:** Red gradient button with "HABLEMOS"
- **Best for:** Fridays
- **Expected Conversion:** 10-20 inquiries/week

### Template C: TESTIMONIO
- **Purpose:** Social proof, build trust, emotional connection
- **Ratio:** 1080 x 1350px (vertical)
- **Layout:** Customer avatar + Quote + Before/After stats + Body + Rating
- **Metrics:** Cyan colored numbers, emphasize results
- **CTA:** "QUIERO MI TRANSFORMACIÓN"
- **Best for:** Sundays
- **Expected Engagement:** 200-400 likes, emotional comments

### BONUS Template D: CAROUSEL
- **5-7 slides** of 1080 x 1350px
- **Slide 1:** Cover (hook)
- **Slides 2-5:** Educational content (one per zone)
- **Slide 6:** Final CTA
- **Color coding:** Each zone has unique color (#18E6D8, #FFB020, #ff3a4b, #9D7BFF)
- **Best for:** Thursdays or Fridays

### BONUS Template E: REELS
- **Dimensions:** 1080 x 1920px (with safe zones)
- **Duration:** 15-30 seconds
- **Format:** Hook → Content → Data → CTA
- **Must haves:** CC/Subtitles, music, clear CTA
- **Best for:** Wednesdays
- **Algorithm boost:** 3-5x more reach than static posts
