GETTING STARTED
================================================================================

Quick Start (5 minutes):
  1. cd frontend
  2. python -m venv venv
  3. .\venv\Scripts\activate
  4. pip install -r requirements.txt
  5. streamlit run app.py

Usage Workflow:
  1. Complete athlete profile
  2. Upload .FIT file from Garmin
  3. Click "Generate Report"
  4. Review charts and metrics
  5. Export PDF or CSV

Documentation Entry Points:

CUSTOMIZATION
================================================================================

Easy Changes (config.py):
  • Colors: Edit COLOR_* constants
  • File size limit: Change MAX_FILE_SIZE_MB
  • Sports options: Modify SPORTS list
  • Validation ranges: Update min/max values
  • Zone definitions: Adjust HR/POWER_ZONES

Moderate Changes (app.py):
  • Add form fields: Extend athlete_profile_form
  • New charts: Create and add chart functions
  • Export formats: Add export buttons
