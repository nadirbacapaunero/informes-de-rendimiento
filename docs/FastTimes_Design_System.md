## 2️⃣ TIPOGRAFÍA

### Sistema de Fuentes

**Primary Font (Headlines, Bold):** Google Font "Inter" (Moderno, Legible)
- Weights: 600, 700, 800
- Para: Títulos, CTAs, Emphasis

**Secondary Font (Body):** Google Font "Inter" (Consistencia)
- Weights: 400, 500
- Para: Descripción, Copy, Body text

**Monospace (Datos/Números):** "Courier Prime" o "IBM Plex Mono"
- Para: Números, tiempos, datos métricos
- Ejemplo: "3:45 min/km", "320+ PBs"

### Reglas de Tipografía

✅ **DO:**
- Usar máx. 2 tamaños de fuente por sección
- Mantener line-height ≥1.5 para legibilidad
- Usar weight 600+ para destaque
- Limitar ancho de texto a 600px (desktop)

❌ **DON'T:**
- Texto menor a 14px en body
- Más de 3 niveles de jerarquía por sección
- Usar color + tamaño sin variación de weight
- Text-transform: uppercase en body text

## 3️⃣ SPACING & SIZING SYSTEM

### Base Spacing Unit: 8px
Todos los espacios son múltiplos de 8px para consistencia.

```
xs    → 4px    (border radius, muy pequeño)
sm    → 8px    (padding botones pequeños)
md    → 16px   (padding estándar)
lg    → 24px   (margin entre secciones)
xl    → 32px   (margin grandes)
2xl   → 48px   (hero section padding)
3xl   → 64px   (mega spacing)
```

### Padding Estándar (Interna)

```
BOTONES
  Padding Horizontal: 24px
  Padding Vertical:   12px
  Border Radius:      6px
  Tamaño min/tap:     44px (touch target)

CARDS
  Padding:            16px (mobile), 24px (desktop)
  Border Radius:      12px
  Margin Bottom:      16px
  Box Shadow:         0 2px 8px rgba(0,0,0,0.1)

POSTS INSTAGRAM
  Padding Header:     16px (profile pic + username)
  Padding Body:       0 (fullbleed image)
  Padding Caption:    16px
  Padding Comments:   16px
  
STORIES
  Safe Zone (top):    60px (status bar)
  Safe Zone (bottom): 80px (gesture bar)
  Content Area:       1080x1920px - safe zones
```

### Márgenes Estándar (Externa)

```
SECCIONES EN WEB/MOBILE

  Margin Vertical:    24px entre secciones
  Max Content Width:  1200px (desktop)

  Gap entre posts:    8px (Instagram standard)
```

## 7️⃣ REGLAS DE CONSISTENCIA VISUAL

### Elementos Obligatorios en CADA POST

1. **Logo/Watermark**
   - Esquina inferior derecha
   - Opacidad: 40%
   - Tamaño: 60 x 60px
   - Posición: 16px desde bordes

2. **Disciplina Badge**
   - Esquina superior izquierda
   - Color por disciplina (rojo/azul/turquesa/púrpura)
   - Tamaño: 80 x 30px
   - Ejemplo: "🏃 RUNNING"

3. **Color Primario**
   - Al menos un elemento #E63946 en cada post
   - Puede ser: Headline, botón, línea, o badge
   - Mínimo 1 x 1cm en impresión (relevante)

4. **CTA Visible**
   - Siempre un CTA claro: "Comenta", "Guarda", "Link en bio", "DM"
   - Mínimo 16px
   - Color rojo o blanco-con-fondo-rojo

### Prohibido (Anti-patterns)

❌ Emojis como iconos principales  
❌ Usar más de 3 colores por post  
❌ Texto menor a 14px en captions  
❌ Más de 2 fuentes diferentes  
❌ Usar degradados sin propósito  
❌ Posts sin CTA  
❌ Inconsistencia en logo/brand  
❌ Tipografía en perspectiva (skew)  
❌ Sombras "caóticas" (múltiples)  
❌ Efectos que no comunican nada

Antes de publicar cualquier post, verifica:

- [ ] ¿Incluye al menos 1 color rojo (#E63946)?
- [ ] ¿El logo está en posición (40% opacity)?
- [ ] ¿La disciplina badge está presente?
- [ ] ¿Hay suficiente contrast? (4.5:1 mínimo)
- [ ] ¿La tipografía es legible en mobile?
- [ ] ¿Las sombras son consistentes?
- [ ] ¿Tiene emoji estratégico? (máx 3)
- [ ] ¿El espaciado es múltiplo de 8px?
- [ ] ¿Todas las imágenes son 1080px mín?
- [ ] ¿La paleta es coherente con template?
- [ ] ¿Respeta safe zones de Instagram?
